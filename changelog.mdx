<Update label="June 2026" description="v1.5" tags={["New", "Improved"]}>
  **New — Slack interactive messages.** Appy agents running in Slack can now send buttons, dropdowns, and other interactive controls directly in the conversation. Click a button and the agent picks up right where it left off — no need to retype your choice.

  **New — Undo/redo in the chat input.** Cmd+Z and Cmd+Shift+Z now work in the task chat input, making it easier to fix a message before sending.

  **New — File uploads in the task chat.** HTML, CSS, JS, and JSON files can now be attached directly in the chat input, so agents can read and work with your code files without any workarounds.

  **New — Screenshot tool.** Agents can now capture full-page screenshots of any public URL and deliver them as files in their response. Useful for visual QA, design reviews, and web monitoring.

  **New — Plan switching for credit-based accounts.** Credit subscribers can now upgrade or downgrade their plan directly from the billing page without going through a new checkout flow.

  **New — Credit usage history on the billing page.** A paginated table on the billing page now shows credits consumed per execution — date, task name, who started it, and credits used.

  **New — Integration setup during onboarding.** New users are now offered a curated integrations picker (Gmail, Slack, Notion, Linear, and more) immediately after signing up, before billing — so your first agent can hit the ground running.

  **New — Credit breakdown on the dashboard.** The dashboard now shows your platform credits and reward credits separately, so you always know exactly what’s available and where it came from.

  **New — Agents page filters on mobile.** The activity and agents page filters no longer squish on small screens. They wrap cleanly below the title on mobile and stay inline on desktop.

  **Improved — SEO metrics are more reliable.** The Moz SEO tool now returns partial results instead of failing entirely when one data source is unavailable.

  **Improved — Connecting integrations from the app is now smoother.** The OAuth connect dialog on the integrations page no longer gets stuck on a spinner after you authenticate. Connections register immediately and the page updates without a manual refresh.

  **Improved — Execution pauses gracefully on low credits.** Instead of failing mid-task when credits run out, agents now pause and notify you via your active channel (Slack, Teams, or Telegram). Resume by replying after topping up — work in progress is preserved.

  **Improved — Agents see files attached to earlier Slack messages.** When you @mention Appy in an existing Slack thread, any files shared in that thread are now automatically available to the agent at the start of the task.
</Update>

<Update label="May 2026" description="v1.5" tags={["Improved"]}>
  **Improved — Slack links render correctly inside bold text.** Bold markdown around links no longer breaks the clickable link formatting in Slack.

  **Improved — Margot (Graphic Designer) agent added.** Margot, a specialist graphic designer, is now part of the built-in agent roster.

  **Improved — Hamilton (Web Design) agent added.** Hamilton, a specialist website designer, is now part of the built-in agent roster.
</Update>

<Update label="June 2026" description="v1.4" tags={["New", "Improved"]}>
  **New — Scheduled Triggers.** Agents can now run on a recurring schedule. Set up daily digests, weekly reports, or any cron-based cadence directly from the agent configuration panel. No code required.

  **Improved — Grain Meeting Synthesis.** The Grain integration now extracts action items as structured tasks, not just transcript summaries. Output is cleaner and maps directly to Linear issues when the Linear integration is active.
</Update>

<Update label="May 2026" description="v1.3" tags={["New", "Improved", "Integration"]}>
  **New — Knowledge Base File Support.** Agents can now reference uploaded PDFs, Word docs, and plain text files as persistent knowledge. Files uploaded to `/lib/knowledge/` are automatically indexed and available across executions.

  **New — Microsoft Teams Integration.** Appy agents are now available natively in Microsoft Teams. Install from the Teams app directory and interact with Violet and your agent team in any channel or DM.

  **Improved — Agent Trigger Reliability.** Integration triggers (GitHub, Slack, webhooks) now retry on transient failures with exponential backoff. Trigger delivery success rate improved from 94% to 99.6%.
</Update>

<Update label="April 2026" description="v1.2" tags={["New", "Improved"]}>
  **New — Multi-Agent Coordination.** Violet can now delegate to multiple specialist agents in parallel and synthesize their outputs into a single deliverable. Tasks that previously required sequential handoffs now complete significantly faster.

  **New — InstantDB Agent Database.** Each agent now has access to a private structured database. Agents can read, write, and query data across executions — enabling stateful workflows like CRM tracking, drip sequences, and ongoing research projects.

  **Improved — Onboarding Flow.** First-time setup now walks through connecting your first integration, hiring your first specialist agent, and running a sample task. Median time-to-first-value dropped from 18 minutes to 6 minutes.
</Update>
