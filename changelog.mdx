<Update label="June 2026" description="v1.4">
  **New: Scheduled Triggers** — Agents can now run on a recurring schedule. Set up daily digests, weekly reports, or any cron-based cadence directly from the agent configuration panel. No code required.

  **Improved: Grain Meeting Synthesis** — The Grain integration now extracts action items as structured tasks, not just transcript summaries. Output is cleaner and maps directly to Linear issues when the Linear integration is active.

  **Fixed: Slack DM delivery on long outputs** — Messages exceeding Slack's block limit were silently truncated. Long agent outputs are now split and delivered as threaded replies.
</Update>

<Update label="May 2026" description="v1.3">
  **New: Knowledge Base File Support** — Agents can now reference uploaded PDFs, Word docs, and plain text files as persistent knowledge. Files uploaded to `/lib/knowledge/` are automatically indexed and available across executions.

  **New: Microsoft Teams Integration** — Appy agents are now available natively in Microsoft Teams. Install from the Teams app directory and interact with Violet and your agent team in any channel or DM.

  **Improved: Agent Trigger Reliability** — Integration triggers (GitHub, Slack, webhooks) now retry on transient failures with exponential backoff. Trigger delivery success rate improved from 94% to 99.6%.
</Update>

<Update label="April 2026" description="v1.2">
  **New: Multi-Agent Coordination** — Violet can now delegate to multiple specialist agents in parallel and synthesize their outputs into a single deliverable. Tasks that previously required sequential handoffs now complete significantly faster.

  **New: InstantDB Agent Database** — Each agent now has access to a private structured database. Agents can read, write, and query data across executions — enabling stateful workflows like CRM tracking, drip sequences, and ongoing research projects.

  **Improved: Onboarding Flow** — First-time setup now walks through connecting your first integration, hiring your first specialist agent, and running a sample task. Median time-to-first-value dropped from 18 minutes to 6 minutes.

  **Fixed: Artifact Download on Safari** — Artifact files (PDFs, DOCX, XLSX) generated by agents were failing to download on Safari 17. This is resolved.
</Update>
