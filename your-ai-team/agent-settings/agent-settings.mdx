Agent settings are the core properties of an individual agent. This is where you configure how a specific agent operates.

## What agent settings cover

Agent settings control the specific properties of one agent:

- Agent name and identity
- AI model selection
- Operational parameters

These settings are specific to one agent, not workspace-wide. Each agent can have different settings based on their role.

## Configuring agent settings

You access agent settings through the Appy.AI web platform.

Each agent has their own settings page where you can adjust their configuration.

Changes you make to one agent don't affect your other agents.

## Key settings

**Agent name**: What your agent is called in Slack and throughout your workspace.

**AI model selection**: Which AI model powers this agent. Different models have different capabilities and costs.

**Operational parameters**: How the agent behaves, including response style, decision-making approach, and other fine-tuning options.

<Info>Most users don't need to change agent settings. The defaults work well for typical use. These are for power users who want fine-grained control.</Info>
